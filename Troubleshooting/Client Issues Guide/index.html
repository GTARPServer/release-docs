    Client Issues Guide

    Could not find game executable¶

    Locate the CitizenFX.ini file within your FiveM Application Data and make sure it is pointing to the correct path. Open the file with a text editor such as Notepad and edit the path to your GTA V installation if necessary.

    If you continue to run into issues we recommend contacting CFX/FiveM

    FiveM is already installed error¶

    After you have installed FiveM, you don't need the downloaded FiveM.exe file anymore. Feel free to delete it.

    Use the shortcut from your Windows start menu to launch FiveM

    Game cache outadated¶

    If you receive this error, press Yes if the dialog asks Do you wish to continue

    If you get an error like DLC files are missing, make sure your game is up to date by following the instructions in the dialog box. Also make sure your CitizenFX.ini points to the correct GTA V installation. You can find that file within your FiveM Application Data.

    Error generating ROS entitlement token¶

    Certain antivirus programs block FiveM. Try disabling or uninstalling your antivirus software and try again or contact CFX/FiveM.

    Error loading component xyz.dll¶

    Delete caches.xml from your FiveM application data folder. If there is no such file, delete the entire application data folder and run FiveM again.

    For adhesive.dll see below

    Error loading component adhesive.dll¶

    Your Windows 10 Installation is out of date. Update to at least Windows 10 1703 and try again.

    To check your current version of Windows press Win+R type winver and press Enter

    Opening database (privcache:/) failed:¶

    IO Error: could not lock file

    Your privcache got corrupted. Delete the "priv" folder from your cache folder located in your FiveM application data

    Entry Point Not Found¶

    Delete v8.dll, v8_libbase.dll, and v8_libplatform.dll if available in C:\Windows\system32\ directory. These are leftover files from other applications that incorrectly use system32 to place these files. FiveM loads dll's from system32 first, resulting in these incorrect dll's being loaded

    Moving of xyz.exe failed (err = 32)¶

    First check your task manager for existing FiveM related processes; if you see them, close the. If that doesn't fix the issue, you may want to contact CFX/FiveM as it's likely related to your antivirus.

    Stuck on 'We're getting there and it will be worth the wait'¶

    If this occurs, it's typically related to antivirus settings. You may want to contact CFX/FiveM regarding your antivirus.

    Stuck on a rotating splash screen¶

    Delete %appdata%\citizenfx\ros_id.dat and %localappdata%\digitalentitlements

    Stuck on a black screen¶

    This most often occurs with certain NVIDIA drivers. Stay patient and give it a minute or so to load, and it'll often succeed. You may also want to update your GPU drivers if available.

    Stuck on a colored background but no menu¶

    This happens typically on older AMD laptop GPUs. Unfortunately, this is caused by CEF and not FiveM.

    There is a potential workaround on the CFX forums, but until CEF fixes the root issue, these may not work permanently.

    FiveM uninstalled itself after running¶

    This is most likely your antivirus software. You may want to submit a support ticket about your antivirus here.

    I have an NVIDIA GPU and FiveM hanged¶

    FiveM hangs when using NVIDIA GPUs as of recently are not uncommon (and especially on the most recent drivers). As of now, CFX are trying to figure out how to resolve hangs for NVIDIA but they're requesting help from users. You can collect information to help them, and us!

    Do not close any processes

    It's very important that you do not close any processes while following these steps, or you will not be able to capture the required information to help CFX/FiveM

    1. Press Win+R and type taskmgr and then press
    2. Once Task Manager opens, go into the Details tab
    3. Find FiveM_GTAProcess.exe, right click it, and click on Create dump file
    4. Wait until the process is dumped, it will be created in the following directory: C:\Users\<your_name>\AppData\Local\Temp\FiveM_GTAProcess.DMP
    5. Click Open File Location and zip up the following file FiveM_GTAProcess.DMP by right clicking it, followed by clicking Send To -> Compressed (zipped) folder on the context menu
    6. Contact CFX Support
